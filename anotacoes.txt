PHP - iMasters

# Passo a Passo Inicial:
- Criar pasta do projeto e acessar pelo prompt.
- Iniciar projeto com composer.
    composer init
- Adicionar autoload.
    composer dumpautoload
- Adicionar o projeto no GitHub.
- Adicionar o ".gitignore".
- Iniciar o servidor PHP.
    php -S localhost:8000

# Anotações
- Post base "https://imasters.com.br/desenvolvimento/criando-planilhas-em-php-com-o-phpspreadsheet/".
- Instalar "PHPSpreadSheet".
    composer require phpoffice/phpspreadsheet
- Rodar arquivo para criar Excel.
    php spreadsheet.php
